# C# MiniGames Collection 🎮

A collection of small games and interactive applications built with **C#** and **.NET WinForms**.

This repository was created as part of my learning journey in C# programming. Each project is a standalone mini-game or simulation, located in its own folder.

## 🚀 How to Run

Each project is a separate **.NET** application.

### Recommended ways to run:

1. **Using Visual Studio / Rider / VS Code**
   - Open the desired game folder
   - Open the `.sln` or `.csproj` file
   - Press **Run** (F5)

2. **Using Terminal / Command Line**
   ```bash
   cd Snake
   dotnet run
  Or to build first:
  Bashdotnet build
  dotnet run --configuration Release
